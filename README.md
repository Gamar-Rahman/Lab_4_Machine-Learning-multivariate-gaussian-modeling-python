# Multivariate Gaussian Modeling in Python

This project explores multivariate normal distributions, covariance structures, and density estimation using NumPy and SciPy.

## Key Concepts Implemented

- Sampling from Multivariate Normal Distribution
- Covariance Matrix Estimation
- Isotropic vs Diagonal vs Full Covariance
- Density Computation on a Grid
- Probabilistic Modeling

## Implemented Functions

- `sample_multivariate_normal`
- `comp_CovList`
- `comp_density_grid`

## Tools Used

- Python
- NumPy
- SciPy
- Matplotlib

## Why This Matters

Understanding covariance structures and multivariate probability distributions is fundamental for:

- Machine Learning
- Anomaly Detection
- Gaussian Mixture Models
- Bayesian Modeling
- Statistical Pattern Recognition
